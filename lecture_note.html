GenAI Note-Taker

As I got to college, I realized that the coursework was very fast-paced and rigourous. Everyday I would wake up, go to class, take notes, do homework, and the cycle never seemed to stop. One of the most cruicial aspects of class for me was to take notes. Notes were very important, as they were often key resources to study for any test or exam. I took notes for almost every class, but due to the limited time and the fast-pace I realized that takings notes and processing the information provided from the professor was difficult. That is why I thought about making this project. To assist me and others like me in taking notes and actually understaind what the coursework, instead of just writing in class for the whole time.

To build a script that can take notes, I had to start with getting a transcript of the audio itself. To do this I used Google Cloud Services API that will used a audio recording sumbited by me to then process it and give me a transcript in return. Now that I received the transcript, I tested it on various audio files, to see its accuracy. Once I felt that the process worked good enough, I proceeded to get Open AI's API to send in the transcript as a user input and use generative AI to create categorized notes for me. To finish this whole project off, I made the input of the files linked to a cloud storage system from Google Cloud Services.

Some future upgrades I would make to this project include: a user interface, easier cloud file upload and synchonization to the user interface.
